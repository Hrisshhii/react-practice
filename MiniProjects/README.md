## üìÇ Project List

### 1. Counter
A simple counter with increment, decrement, with keyboard shortcuts and reset functionality.  
- Concepts: useState, useEffect hook, event handling, component state updates

### 2. Todo App
Persistent todo app with edit, delete, complete, animations, and localStorage.  
- Concepts: CRUD, localStorage, controlled inputs, refs, conditional UI, immutability

### 3. Meals App
Meal explorer using TheMealDB API with categories, search, random meal, and modal details.
- Concepts: Axios, API fetching, multiple useEffect hooks, component composition, modals

### 4. Calculator
Fully functional calculator with keyboard input support and error handling.
- Concepts: useState, useEffect, mathjs integration, event listeners, expression evaluation

### 5. Toggle Background
Dark/light mode switch with persistence and keyboard toggle.
- Concepts: Theme state, localStorage, conditional styling, keyboard shortcuts

### 6. Hidden Search Bar
Animated expandable search bar with focus control and outside click detection.
- Concepts: useRef, useEffect, DOM events, conditional UI, UX interactions

### 7. PokeDex (Pokemon Slider)
Auto-sliding Pok√©mon carousel with manual controls and animations.
- Concepts: useState, useEffect (intervals), carousel logic, dynamic class renderin

### 8. Accordion
Accessible accordion component with keyboard navigation and smooth toggling.
- Concepts: useState, conditional rendering, keyboard events, accessibility, array mapping

### 9. Form Validation
Real-time validated registration form with regex checks and keyboard shortcuts.
- Concepts: Controlled components, form validation, regex, conditional rendering, useEffect

### 10. Ecommerce Filtering
Product listing with search, multi-filtering, and sorting (price & rating).
- Concepts: Array filtering, sorting, controlled inputs, derived state, UI state management

### 11. Weather App
City-based weather search using OpenWeather API with geocoding and error handling.
- Concepts: Async/await, environment variables, API chaining, loading & error states

### 12. Music Player
Custom audio player with playlist, progress bar, seeking, and volume control.
- Concepts: useRef, useState, audio API, progress tracking, media controls
